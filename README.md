# Pre-School Management System

## Project Overview
A web-based Pre-School Management System developed for Merry Kids International Montessori
to streamline student admissions, attendance, communication, and financial management
through a centralized digital platform.

## Client
Merry Kids International Montessori School

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Server: WAMP
